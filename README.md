## Project Overview

Back 2 Back Studios is a responsive salon website for a Jacksonville, FL salon. Clients can view services and pricing and submit appointment booking requests online. Built with HTML, CSS, and JavaScript using localStorage to save booking data.

---

## Features

- Home page with hero section and call-to-action
- Services section with 6 treatments and pricing
- Booking form with confirmation message
- localStorage saves booking requests between sessions
- Fully responsive – works on mobile, tablet, and desktop
- Mobile hamburger navigation menu

3. No installation needed
